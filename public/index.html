Database Schema

Students

  • id UUID (Primary Key)
  • student_id VARCHAR(20)
  • first_name VARCHAR(50)
  • last_name VARCHAR(50)
  • email VARCHAR(100)
  • phone VARCHAR(15)
  • program VARCHAR(100)
  • year INTEGER
  • created_at TIMESTAMP

Courses

  • id UUID (Primary Key)
  • course_code VARCHAR(20)
  • title VARCHAR(100)
  • description TEXT
  • credits INTEGER
  • department VARCHAR(100)
  • faculty_id UUID (Foreign Key)
  • semester VARCHAR(20)

Grades

  • id UUID (Primary Key)
  • student_id UUID (Foreign Key)
  • course_id UUID (Foreign Key)
  • grade VARCHAR(2)
  • score DECIMAL(5,2)
  • semester VARCHAR(20)
  • academic_year VARCHAR(10)
